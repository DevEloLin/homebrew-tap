# develolin/tap

Homebrew tap for [EvoClaw](https://github.com/DevEloLin/evoclaw) — self-evolving local-first personal AI agent.

## Install

```bash
brew install develolin/tap/evoclaw
```

Or tap first then install:

```bash
brew tap develolin/tap
brew install evoclaw
```

## Upgrade

```bash
brew upgrade evoclaw
```

## Uninstall

```bash
brew uninstall evoclaw
brew untap develolin/tap
```

## Notes

- This tap tracks **stable releases only**. Beta versions (`-beta`, `-rc`) are not published here.
- Each stable release is automatically pushed to this tap by the EvoClaw release pipeline.
- Binaries are provided for macOS Apple Silicon (arm64).
- For Linux and Windows, download directly from the [Releases page](https://github.com/DevEloLin/evoclaw/releases).
